## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/bobthesiddharth/todolist-SpringBoot.git
cd todolist-SpringBoot

# Add DB credentials to application.properties
# Run the application
./mvnw spring-boot:run
